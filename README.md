# A Dual-Hidden Layer Neural Network with Delta-Bar-Delta Adaptive Learning Approach to Predicting the Approval or Rejection of a Loan Application


## Overview
This project implements a loan approval prediction system using a deep (2 hidden layers) feedforward neural network trained with the Delta-Bar-Delta with Momentum algorithm. The model learns from preprocessed loan application data to classify loan outcomes as Approved or Rejected using encoded inputs and a bounded sigmoid output target. The pipeline includes preliminary data checking, preprocessing, training, evaluation, and threshold tuning. The main implementation demonstrates how a dual-hidden-layer neural network can be used for predicting rejection or approval of loan applications.

## File structure
- `data/`
  - `data.csv` — raw loan application dataset used for preprocessing.
- `data_preprocess.py` — preprocesses `data/data.csv`, encodes categorical inputs, scales continuous features to [0.1, 0.9], performs an 80/20 train/test split, and saves `X_train.csv`, `Y_train.csv`, `X_test.csv`, `Y_test.csv`.
- `deep_nn.py` — trains a deep neural network on the preprocessed train set, evaluates the model, computes threshold metrics, diagnoses overfitting, and saves `lc_phase2b.csv` and `results_phase2b.csv`.
- `check_data.py` — performs an initial data audit on `data/data.csv` to inspect shape, missing values, class balance, unique values, descriptive statistics, and duplicates.
- `fwb.py` — sample shallow network implementation; included for reference only and not required for the main pipeline.
- `lc_phase2b.csv` — learning curve history output from `deep_nn.py`.
- `results_phase2b.csv` — threshold tuning and final evaluation results output from `deep_nn.py`.

## Requirements
- Python 3.8 or newer
- `pandas`
- `numpy`

Install dependencies with:

```bash
pip install -r requirements.txt
```

Or directly:

```bash
pip install pandas numpy
```

## What to run first, second, etc.
1. `python check_data.py`
   - Optional but recommended for dataset auditing before preprocessing.
2. `python data_preprocess.py`
   - Creates the preprocessed training and test CSV files required by `deep_nn.py`.
3. `python deep_nn.py`
   - Trains the deep network and produces evaluation results and output CSV files.

## Notes
- `fwb.py` is a sample implementation of a shallow network and is not part of the main deep network pipeline.
- The pipeline depends on the intermediate CSV files generated by `data_preprocess.py`.
- Output files from the main training run are `lc_phase2b.csv` and `results_phase2b.csv`.
